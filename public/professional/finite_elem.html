Computationally Solving Nonlinear Membranes with Plane Stress Condition

This project was completed as part of my Masters curriculum for the class "261B: Computational Mechanics of Solids and Structures." Membrane theory, plane stress condition, and the neohookean constitutive laws were used in computation. The coding itself was done in MATLAB R2015a Student Version. The final report is linked here and source code is linked here

The overall purpose of the report was to develop a nonlinear solver for deformed membranes while enforceing the plane stress condition. The developed code was tested on several geometries, some of which are shown below.
